# DeepSeek CLI Chat Tool

一个基于 DeepSeek API 的命令行聊天工具，支持多会话管理、流式输出和本地 JSON 持久化。

---

## ✨ 功能特性

- 多会话管理（支持创建 / 切换 / 删除对话）
- 对话历史本地持久化（JSON 文件）
- 流式输出（实时打印模型回复）
- 自动保存（每轮对话后写入磁盘，防止数据丢失）
- 基础命令系统（exit / save / new / delete）

---

## 📁 项目结构

├── deepseek.py # 主程序

├── conversations/ # 对话存储目录（初次使用将自动生成）

│ ├── xxx.json

│ └── ...

├── .env # API Key（需自行创建）

└── README.md

---

## ⚙️ 环境要求

- Python 3.8+

- 已安装依赖：

```
bash
pip install openai python-dotenv
```

---

## 🔑 配置 API Key

在项目根目录创建 .env 文件：
```
ini
DEEPSEEK_API_KEY=sk-xxxxxx
```

---

## 🚀 使用方法

运行程序：
```
python
deepseek.py
```

---

## 💬 基本操作

启动后进入对话选择界面：

输入序号 → 选择已有对话

输入名称 → 创建新对话


对话命令:

命令:说明

exit:	保存并退出程序

save：手动保存当前对话

new：保存当前对话并切换会话

delete：删除当前对话

---

## 🧠 工作原理（简要）
所有对话以 JSON 格式存储在 conversations/ 目录下

每个对话对应一个文件

每轮请求将完整对话历史发送给模型

使用流式 API 实现逐字输出

---

## ⚠️ 注意事项
对话历史会持续增长，可能导致：

token 消耗增加

响应变慢
