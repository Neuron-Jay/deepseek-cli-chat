# 🧠 Jay_MediSift

> A lightweight AI-powered literature processing toolkit
> Built by **Neuron_Jay**

---

## 📌 项目简介

**Jay_MediSift** 是一个面向医学学习与科研的轻量级AI工具，旨在提升文献阅读效率、降低信息处理成本。

本项目围绕“**文本读取 → AI分析 → 结构化输出**”这一核心流程构建，支持单篇文献精读与批量处理两种模式，并提供基础对话能力。

---

## 🚀 核心功能

### 1️⃣ `deep_agent`（核心功能）

用于**单篇文献分析**的核心模块。

#### 功能特点：

* 支持 `.txt / .docx / .pdf` 文件读取
* 可手动选择分析方式（skill）
* 自动生成结构化分析结果
* 自动保存为 `.md` 文件

#### 当前支持的 Skills：

* **结构化总结（summary）**

  * 输出：背景 / 问题 / 方法 / 结果 / 意义

* **解释文本（explain）**

  * 输出：类似授课风格的通俗讲解

* **提取关键词（keywords）**

  * 输出：关键概念 + 简要解释

---

### 2️⃣ `batch_deep_agent`

在 `deep_agent` 基础上扩展的**批量处理模块**

#### 功能特点：

* 自动遍历指定文件夹中的所有文献
* 对每篇文献执行全部或指定 skills
* 自动分类输出到不同结果目录

#### 使用场景：

* 批量处理文献（10+ 篇）
* 快速建立文献初筛结果
* 构建知识库素材

---

### 3️⃣ `deep_chat`

基础AI对话模块

#### 功能特点：

* 支持与 DeepSeek 模型进行对话
* 可在源码中自定义模型
* 可扩展为多模型选择系统（未来版本）

## ⚙️ 使用前准备（必须）

在运行本项目之前，请确保完成以下步骤：

---

### ✅ 1. 安装依赖库

```bash
pip install openai python-docx pymupdf python-dotenv
```

---

### ✅ 2. 配置 API Key

在项目根目录创建 `.env` 文件：

```
DEEPSEEK_API_KEY=你的API_KEY
```

---

### ✅ 3. 文献存放目录（自动生成）

在正常情况下，**首次运行脚本后，系统会自动创建以下两个文件夹：**

text_file_repository/
paper_batch/


---

#### 📂 text_file_repository
用于 **deep_agent（单篇分析）**

👉 将需要单篇分析的文献放入该文件夹

---

#### 📂 paper_batch
用于 **batch_deep_agent（批量处理）**

👉 将需要批量处理的文献放入该文件夹

---

#### 🤔 若未自动生成

在项目根目录手动创建以下文件夹：

text_file_repository
paper_batch

---

### ✅ 4. 关于数据安全

项目已配置 `.gitignore`：

* 所有本地文献 ✔️
* API Key ✔️
* 输出结果 ✔️

**这可以确保您的个人数据不会被上传到 GitHub**

## 🧪 使用方法

---

### ▶ 单篇分析（deep_agent）

```bash
python deep_agent.py
```

流程：

1. 选择文献
2. 选择分析方式（skill）
3. 自动输出结果

---

### ▶ 批量处理（batch_deep_agent）

```bash
python batch_deep_agent.py
```

流程：

1. 将多篇文献放入 `text_file_repository`
2. 运行脚本
3. 等待自动处理完成
4. 在输出文件夹中查看结果

---

### ▶ 对话功能（deep_chat）

运行对应脚本后即可进入交互式对话模式。

## 💡 项目设计理念

Jay_MediSift 并不是一个“全自动 Agent”，而是：

> **Human-in-the-loop AI 工具**

* 人类负责决策（选择需要分析的文献）
* AI负责执行（处理文本）

---

## ⚖️ 为什么不完全自动化？

在医学文献场景中：

* 信息密度高
* 错误成本高
* 上下文依赖强

👉 因此本项目选择：

> **高可靠性优先，而非全自动决策。**
> **换而言之本项目不会对您存放的文献进行自动化筛选，这一步交给您来决定。**
